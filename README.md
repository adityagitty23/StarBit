# 🍽️ StarBit QR Ordering System

A modern QR-based restaurant ordering system built with React, Vite, Tailwind CSS, and Flask.

Customers can scan a QR code, browse the menu, add items to their cart, and place orders directly from their mobile devices without installing any app.

---

## 🚀 Features

### Customer Side
- QR Code Menu Access
- Mobile-First Responsive Design
- Category-Based Menu Navigation
- Add to Cart Functionality
- Quantity Management
- Customer Details Collection
- Cash & Online Payment Options
- Real-Time Order Placement

### Restaurant Side
- Order Management Dashboard
- Accept / Reject Orders
- Update Order Status
- Menu Management
- Restaurant Branding Support
- Customer Order Tracking

---

## 🛠️ Tech Stack

### Frontend

6. Place Order
7. Restaurant Receives Order
